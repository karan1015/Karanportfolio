Project-1 Data Science Project

Sound Based Birds Classification Using YAMNet

The project, "Sound-Based Birds Classification using YAMNET," aims to utilize YAMNET's deep learning capabilities for accurately classifying bird species based on their vocalizations. This automated system provides a non-invasive and efficient means of monitoring avian populations, contributing to biodiversity conservation efforts. By leveraging sound patterns, the project enhances ecological research, showcasing the potential of technology in advancing wildlife monitoring and conservation practices.

Import Libraries

YAMNet is a pre-trained deep neural network that can predict audio events from 521 classes, such as laughter, barking, or a siren. In this tutorial you will learn how to:

  • Load and use the YAMNet model for inference.
  • Build a new model using the YAMNet embeddings to classify birds sound
  • Evaluate and export your model

About YAMNet

YAMNet, from Google's Magenta project, excels at recognizing bird vocalizations, being a potent pre-trained audio classification model. Trained on the extensive AudioSet dataset, it offers an efficient solution for ecological studies in bird sound classification.

Data Set

The code downloads a birds dataset from a specified URL, extracts it locally, and creates a pandas DataFrame (df) by reading the metadata.csv file. The head() function displays the initial rows of the dataset for a quick overview.

Explore The Data

The variable data_dir signifies the directory path ('./dataset/small_birds_dataset') where a collection of small bird images is stored for further analysis. The bird_code_to_name dictionary establishes a correlation between specific bird codes and their respective names, streamlining the process of interpreting coded labels within the dataset.

  • Data Directory
  • Bird Code Mapping
  • Bird Images

Begin by preprocessing the bird sound dataset, splitting it into training and validation sets for evaluation during training. Define the model architecture for sound classification, compile it with appropriate optimizer and loss functions, and initiate the training process.

Snake Game - Classic Arcade Fun with Python & Pygame

Project Link: Click Here



Step into the world of retro gaming with my Python-based Snake Game! Built using the Pygame library, this project brings back the timeless arcade classic where players guide a snake to collect food, grow in size, and navigate through increasingly challenging levels. But watch out! Hitting the wall or biting your own tail means game over.

Technologies Used

  • Python: A versatile and beginner-friendly programming language used to create the game's core logic, handle game events, and manage snake movement and interactions.
  • Pygame Library: A popular game development library for Python, enabling smooth rendering of graphics, handling user input, and managing real-time game loops.
  • Object-Oriented Programming: Utilized to structure the game’s components efficiently, ensuring the code is modular, maintainable, and scalable.

About Snake Game

  • Snake is a classic arcade game that has been entertaining players since the late 1970s.
  • Originally developed as a simple yet addictive game, its goal is to control a moving snake as it navigates a confined space, collecting food to grow longer.
  • However, the challenge increases as the snake becomes harder to control without colliding with itself or the walls.

Feature

  • Smooth Movement & Controls: Use arrow keys to seamlessly maneuver the snake around the game board.
  • Dynamic Gameplay: As you collect food, the snake grows longer, adding a layer of strategy and reflex to avoid self-collision.
  • Score Tracking: Keep tabs on your high score and challenge yourself to beat your personal best.
  • Play Again Option: After the game ends, instantly restart or quit with simple keypress options.
